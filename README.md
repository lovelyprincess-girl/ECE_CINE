# ECE_CINE

## Description
ECE_CINE est un site de cinéma conçu pour présenter différents films avec leurs rôles, actions, synopsis, et catégories.  
Le projet inclut des maquettes réalisées sur **Figma** pour le design et l’interface utilisateur.

## Fonctionnalités prévues
- Affichage des films avec leur affiche et description
- Gestion des rôles des personnages
- Navigation par catégories 
- Pages interactives pour chaque film
- Maquettes Figma pour le design et la planification

## Technologies
- HTML / CSS / PHP
- JavaScript pour les interactions
- Base de données MySQL (prévue pour stocker les films, utilisateurs, etc.)
- Figma pour les maquettes UI/UX
